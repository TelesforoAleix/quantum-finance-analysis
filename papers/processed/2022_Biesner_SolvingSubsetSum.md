## Abstract summary
This preprint explores the application of quadratic unconstrained binary optimization (QUBO) to solve the subset sum problem, which is central to automating auditing and financial data consistency checks. The authors reformulate the subset sum problem as a QUBO and demonstrate how gradient descent on Hopfield Networks can efficiently find solutions for both synthetic and real-world financial datasets. The work also discusses potential advancements using specialized hardware and quantum algorithms.
## Methodology
The paper presents an empirical approach to solving the subset sum problem, which is NP-hard, using binary optimization techniques with applications in auditing and financial data analysis. The subset sum problem is reformulated as a Quadratic Unconstrained Binary Optimization (QUBO) problem. The authors employ gradient descent on Hopfield Networks to find solutions for both artificial and real financial datasets. The methodology involves converting the subset sum problem into a QUBO, then mapping it to a Hopfield Network where gradient descent updates are applied to minimize the energy function, which corresponds to solving the QUBO. The algorithm is evaluated on multiple configurations of artificial data and real financial data extracted from Deutsche Bank's financial reports.

**Algorithms used:** Hopfield Networks

**Experimental setup:** The experiments were conducted using artificial data generated by uniformly sampling integers within specified ranges and real financial data parsed from Deutsche Bank's financial reports. The Hopfield Network algorithm was implemented and run on an NVIDIA A100 GPU, allowing parallel optimization of multiple independent networks. The algorithm was tested for up to 1e+8 initializations in batches of 1e+4, with a maximum computation time of around 8 minutes per configuration.

**Dataset:** The artificial datasets consist of uniformly sampled integers between specified bounds (Xmin and Xmax) with varying sizes (n) and target sums (T). The real financial datasets include 190 independent columns from Deutsche Bank's financial reports covering quarters from Q1 2019 to Q4 2020, with values ranging up to billions of euros and precise to one cent.
