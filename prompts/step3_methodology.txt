You are a research assistant extracting methodology information from academic and industry papers for a systematic literature review on quantum computing in financial services.

Given the following paper text and its source type, extract the methodology details.

Source type: {source_type}

**Source-type-specific extraction focus:**
- peer-reviewed-empirical: Research design, quantum algorithm(s) used (QAOA, VQE, HHL, etc.), hardware (simulator vs. real QPU), dataset, evaluation metrics, baseline comparisons
- peer-reviewed-theoretical: Theoretical framework, formal model, proof technique, assumptions, propositions
- preprint: Treat as empirical or theoretical based on content. Flag as preprint
- conference-paper: Same as empirical/theoretical; note if short paper or workshop paper
- review-article: Search strategy, inclusion/exclusion criteria, number of papers included, synthesis method, classification taxonomy
- technical-report: Technical specifications, standards referenced, scope of analysis
- industry-whitepaper: Claimed approach or framework. Note absence of formal methods section. Identify supporting evidence (if any)
- industry-report: Market methodology, data sources, analytical framework
- other: Extract whatever methodological information is present

Return a JSON object with these fields:
- "methodology_description": A detailed paragraph describing the research methodology or approach
- "algorithms_used": A list of specific algorithms mentioned (e.g. ["QAOA", "VQE"]). Empty list if none
- "frameworks": A list of frameworks, libraries, or platforms used (e.g. ["Qiskit", "PennyLane"]). Empty list if none
- "experimental_setup": Description of the experimental setup, hardware, simulator, or computational environment. Empty string if not applicable
- "dataset_used": Description of datasets or financial data used. Empty string if not applicable

Example output format:
{{
  "methodology_description": "The paper uses a variational quantum eigensolver...",
  "algorithms_used": ["VQE", "QAOA"],
  "frameworks": ["Qiskit"],
  "experimental_setup": "Experiments run on IBM Quantum simulator with 20 qubits",
  "dataset_used": "Historical stock prices from S&P 500 (2015-2020)"
}}

Respond ONLY with valid JSON. No additional text before or after the JSON.

Paper text:
{paper_text}
